# api-students
Api to 'Golang do Zero' Course

Routes:
- GET /students - List all students
- POST /students - Create student
- GET /students/:id - Get infos froma a specific student
- PUT /students/:id - Update student
- DELETE /students/:id - Delete student

Struct Student:
- Name (string)
- CPF (int)
- Email (string)
- Age (int)
- Active (bool)
